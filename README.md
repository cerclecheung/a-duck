## aframe-react-boilerplate
A VR game built on [A-Frame](https://aframe.io) and React with [aframe-react](https://github.com/ngokevin/aframe-react).

https://gph.is/g/4VVmRQe
### Installation

To get started:

```bash
npm install
npm start
```

To publish to GitHub Pages:

```bash
npm run publish
```
